# Deep Research Skill

Conduct thorough multi-step research on any topic: search → read → synthesize → report.

## Requirements
- web-search skill must be available
- browser tool for reading full articles

## Usage
- "research X in depth"
- "give me a full report on X"
- "deep dive into X"
- "what's the current state of X"

## Behavior
1. Run 3-5 web searches from different angles on the topic
2. Read the most relevant full-text sources using browser tool
3. Synthesize findings into a structured report
4. Include sources and key takeaways
5. Flag conflicting information if found
